Documentation Ultimate PS3 Hack Script

 

Introduction

 

Ceci est un ensemble de script permettant de réaliser différentes tâches pour permettre de profiter simplement des différentes possibilités qu'ofre le hack PS3.

Cette documentation a pour but d'orienter l'utilisateur pour faire fonctionner au mieux ces outils. Le sujet concernant cette application se trouve sur cette page.

Ce script est utilisable sur Windows 7 32 et 64 bits ainsi que sur les versions supérieurs (Windows Vista non testé).

Veuillez bien prendre le temps de lire les instructions durant le script car certaines actions nécessitent d'être prudent pour ne pas endommager le matériel.

Notez que je ne pourrai être tenu responsable en cas de domages résultant de l'utilisation de ce logiciel et de tout se qu'il contient.

 

Fonctionnalités

 
  • Mise en place du hack pour les consoles compatibles sous le firmware 4.82 et sous le firmware spécial 4.84. Pour en savoir plus, consultez ce tutoriel.
  • Mise en place du hen pour toutes les consoles sous le firmware 4.82 et sous le firmware spécial 4.84.
 

Bugs connus

 
  • L'utilisation de guillemets dans les entrées utilisateurs fait planter le script.
  • Lorsqu'une sortie console faite par un "echo" est effectuée, cela produit une erreur dans le fichier log. L'encodage en UTF-8 semble être à l'origine de ce problème mais je n'ai pas trouvé comment le résoudre pour l'instant.

Changelog

 

V1.11

  • Quelques changements dans le menu principal du script pour le rendre plus lisible et plus simple à utiliser.

V1.10

  • Grosse refonte du script en général, les dossiers ont été réorganisés, certains bugs corrigés, le projet adapté pour être mis sur Github, etc...
  • Ajout de la possibilité d'utiliser les différents outils pour le firmware spécial 4.84 intégrant la faille webkit du 4.82, ce firmware particulier peut être préparé sur un support USB durant les différents scripts pour être ensuite rapidement installé sur une console disposant d'un firmware inférieur ou égal au firmware officiel 4.84.
  • Ajout du hen PS3 pour toutes consoles sou firmware inférieur ou égal au firmware officiel 4.84, est particulièrement utile pour les consoles ne pouvant pas installer de CFW. Attention, tout comme l'exploit permettant d'installer un CFW pour les consoles compatibles pour les firmwares 4.82 ou 4.84 spécial, il faut d'abord se trouver sous un de ces firmwares pour utiliser le hen.

V1.09

  • Remplacement de Miniweb Server par Caddy Web Server pour corriger certains problèmes.
  • Changement du binaire de Wget vers la version 19.04.

V1.08

  • Le script utilise maintenant PS3Xploit 2.0.
  • Ajout des fonctions de dump de la NAND/NOR et de l'IDPS.
  • Ajout et modification de certains messages.
  • Remaniement du script de flash pour intégrer au mieux les nouvelles possibilité offertent par la version 2.0 de PS3Xploit.
  • Le script de restauration d'un support USB est adapté aux fichiers de la version 2.0 de PS3Xploit.
  • Mise à jour de la documentation pour ajouter des informations sur PyPS3checker.

V1.07

  • Ajout de la liste des modèles nor compatibles lors du choix de l'exploit à lancer. Attention, je rappel que les modèles CECH-25xxx ne sont pas toutes compatibles avec le hack donc bien lire les notes en début de script.

V1.06

  • Création d'un script "preload.bat" qui indique à l'utilisateur comment changer la police de caractères pour en choisir une qui soit compatible avec l'UTF-8. Si celle-ci ne l'est pas, le script s'arrête juste après cette vérification.
  • Modification du raccourci de lancement du script pour qu'il pointe sur le nouveau script "preload.bat".
  • Déplacement de quelques instructions du script "LogHook.bat" vers le script "preload.bat" pour que le tout soit cohérent.

V1.05

  • Optimisation d'une partie du code.

V1.04

  • Ajout de la possibilité de passer toutes la phase de préparation du support USB pour lancer le serveur web hébergeant l'exploit souhaité rapidement. S'il vous plaît, si vous n'êtes pas sure de la compatibilité de la console ou si vous n'êtes pas sure de se que vous faites, veuillez suivre la phase de préparation du support USB pour éviter de bricker votre console.

V1.03

  • Ajout de la possibilité de préparer la clé USB pour installer l'OFW 4.82.
  • Ajout de la possibilité de sélectionner un fichier de CFW 4.82 personnel. Attention, aucune vérification n'est vraiment faite donc soyez prudent avec cette fonctionnalité.
  • Ajout d'un script permettant de supprimer les fichiers copiés par le script de préparation du hack 4.82 d'une clé USB.
  • Ajout d'un script permettant de réinitialiser le script comme s'il venait d'être téléchargé. Notez que si vous avez créé des fichiers ou dossiers non géré pendant le script, ils ne seront pas supprimés.
  • Quelques autres modifications mineurs.
